DOMAIN NAME AND YOUR BROWSER

if you were asked as to what will happen when you enter the domain name in the address bar of a browser? an internet user will surely tell you that web-page of the site will appear but as a developer, you will be expected to analyze how the internet works behind the scene. The domain name madeiternet protocol (IP address) simple to remember because is been replaced with letters human beings already familiar with instead of a numeric value (#.#.#.#). For instance, ipv4 or ipv6 is in 0 and 1(#.#.#.#) it uniquely identifies the computer on the internet and that's what allows computers to talk to each other

And before the advent of DHCP, you will need to get a technician to configure your device for you for it to be able to use internet services. However, the Dynamic Host Configuration Protocol now responsible for allocating IP address to computer automatically. This DHCP comes from the router or some kind of server so any time you on your computer and it will provide you with unique IP

How does my computer know how to find the other IP addresses whenever I type in the www.exanple.com. DNS (Domain Name System}is the one responsible for converting the domain name into correspondence IP addresses so it facilitates this communication between computers

TCP transmission control protocol...this is what will ensure that whatever an internet user requested/packet get to their intended destination. The language of the internet is HTTP (hypertext transfer protocol)
